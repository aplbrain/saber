Overview

Welcome to the SABER MICrONS Eval webapp. This app utilizes SABER to run MICrONS workflows on the cloud.

Workflows are executed through jobs. Each job requires and experiment and a docker image to run. Experiments include task-specific parameters and IO information.

How to run a workflow

  1. Create Experiment.
  2. Create Job. Workflow will execute once job is created.
  3. Download output on jobs page.

Requirements

  • Image must be built on webserver.
  • Output and input S3 buckets must already exist.
  • Workflow can only output a CSV file.
